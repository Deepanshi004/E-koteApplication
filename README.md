# 🔫 E-Kote Weapon Management System

A secure and efficient Android-based Weapon Management System designed to streamline weapon inventory handling, allocation tracking, and administrative management.

The system helps organizations maintain weapon records digitally with improved transparency, security, and real-time monitoring.

---

# 🚀 Features

## 🔐 Authentication & Security
- Secure Login System
- Role-Based Access Control
- Admin & Authorized User Access
- Data Protection & Validation

---

## 🗄️ Weapon Inventory Management
- Add New Weapons
- Update Weapon Information
- Delete Weapon Records
- Search & Filter Weapons
- Track Weapon Availability

---

## 📋 Weapon Allocation System
- Assign Weapons to Personnel
- Track Issued & Returned Weapons
- Maintain Allocation History
- Real-Time Status Updates

---

## 📊 Dashboard & Monitoring
- Weapon Inventory Overview
- Allocation Statistics
- Active Issued Weapons
- Pending Returns Monitoring

---

## 📱 User-Friendly Interface
- Android-Based Application
- Clean & Responsive UI
- Easy Navigation
- Fast Data Access

---
